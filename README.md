# 안암동 맛집 캐치테이블

고려대 주변 맛집 10곳을 골라 확인하고, 예약이랑 리뷰까지 남길 수 있게 만든 웹 페이지입니다. 저희가 고려대학교 학생들의 공강시간을 책임지겠습니다!

## 주요 기능
* **안암동 맛집 TOP 10**: 학생들이 자주 가는 안암동 맛집 10곳 중에서 원하는 곳에 방문해보아요. 여러분들의 만족감 있는 식사를 도와드립니다.
* **오늘의 AI 맛집 비서의 추천**: 매일 달라지는 오늘의 추천 맛집, 결정하는 것이 어렵다면 AI의 추천을 믿어보는 건 어떤가요? AI 맛집 비서에게 오늘의 기분이나 오늘의 날씨를 입력한다면 딱 맞는 맛집을 추천해줄 것입니다.
* **실시간 예약 신청**: 이름, 연락처, 인원 수를 작성하세요! 원하는 시간대를 골라 예약하면 오른쪽 사이드바에서 예약 내역을 바로 확인할 수 있습니다.
* **별점 및 한줄평**: 다녀온 식당에 별점과 리뷰를 남길 수 있고, 다른 사람들이 남긴 평점도 모아서 평균으로 보여줍니다. 다른 손님들이 어떤 이야기를 남겼을지 궁금하진 않으신가요?
